# Template
- This is template for **VANILLA JS** projects you can add here missing technologies that you want

---

### Setup:
- Vite
- JavaScript
- SASS
- Eslint
- Prettier

---

### Getting Started

```bash
npm i
```

### Run dev mode

```bash
npm run dve
```

### Build

```bash
npm run build
```

### Format all files by Prettier

```bash
npm run format
```
