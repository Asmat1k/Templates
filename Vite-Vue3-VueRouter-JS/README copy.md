# Template
- This is template for REACT project you can add here missing technologies that you want

---

### Setup:
- Vite
- React
- React Router Dom
- TypeScript
- Eslint
- Prettier
- Husky & lint-staged

---

### Getting Started

```bash
yarn init-project
```

### Run dev mode

```bash
yarn dev
```

### Build

```bash
yarn build
```
